asteroid.losses.mse module¶

Mean squared error loss functions.
@author : Manuel Pariente, Inria-Nancy
asteroid.losses.mse.nonpit_mse(targets, est_targets)¶

Measure mean square error on a batch.

Supports both tensors with and without source axis.

Parameters

  • targets – torch.Tensor. Expected shape [batch, *]. The batch of training targets.

  • est_targets – torch.Tensor. Expected shape [batch, *]. The batch of target estimates.

Returns

torch.Tensor – with shape [batch]

Examples

>>> import torch
>>> from asteroid.losses import PITLossWrapper
>>> targets = torch.randn(10, 2, 32000)
>>> est_targets = torch.randn(10, 2, 32000)
>>> # nosrc_mse / nonpit_mse support both 'wo_src' and 'w_src'.
>>> loss_func = PITLossWrapper(nosrc_mse, mode='wo_src')
>>> loss = loss_func(targets, est_targets)
>>> print(loss.size())
torch.Size([10])
