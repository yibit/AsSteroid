asteroid.losses.si_sdr module¶

Negative Scale invariant Signal to Distorsion Ratio (SI-SDR) losses.
@author : Manuel Pariente, Inria-Nancy
asteroid.losses.si_sdr.nonpit_neg_sisdr(targets, est_targets, scale_invariant=True)[source]¶
Measure mean negative SI-SDR for a given permutation of source and

their estimates.

Parameters

  • targets (torch.Tensor) – Expected shape [batch, time]. Batch of training targets.

  • est_targets (torch.Tensor) – Expected shape [batch, time]. Batch of target estimates.

  • scale_invariant (bool) – Whether to rescale the estimate to the target.

Returns

torch.Tensor –

with shape [batch].

Batch losses for this permutation.

Examples

>>> import torch
>>> from asteroid.losses import PITLossWrapper
>>> targets = torch.randn(10, 2, 32000)
>>> est_targets = torch.randn(10, 2, 32000)
>>> loss_func = PITLossWrapper(nonpit_neg_sisdr, mode='w_src')
>>> loss = loss_func(targets, est_targets)
>>> print(loss.size())
torch.Size([10])
